비고(Description)

번호 분류 내용 비고
1 요약 Smart Premiere 프로젝트는 PC 기반의 이미지 인식 및 학습 등 인공지능을 응용한 프로젝트로서
웹 프로젝트를 연동한 융합 프로젝트이다.
2 Web Smart premiere-web은 웹상에서 관리할 수 있는 환경을 구현하였다.
3 Controller Smart Premiere C Sharp과 OpenCVsharp 4, ZXing, MariaDB로 개발하였다.
4 Ai Smart Premiere Ai는 Tensorflow, Numpy, sklearn, PIL, pymysql, 자료구조와 알고리즘을 기반으로 개발하였다.
5 IoT Smart Premiere IoT는 Arduino Wifi Rev 2.0와 온습도센서, 충격감지센서, 초음파센서로 개발하였다.
6 Robot Arm Smart Premiere 6dof Robot Arm은 Arduino Uno R3와 6dof프레임구조, MG-996R 모터 등으로 개발하였다.
