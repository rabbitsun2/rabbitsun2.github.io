  1. Home
  2. Feature
  3. Linked with robot arm(로봇 팔과 연동)
스마트 프리미어의 주요 기능입니다.

1. 설명
  • 예를 들면, 6축 로봇팔과 연동하여 PC기반에서 원격으로 제어할 수 있습니다.
    (For example, it can be controlled remotely on a PC-based basis in conjunction with a 6-axis robot arm.)
    이해를 돕기 위해서 6축 로봇팔 축소판을 구현하였습니다.
    (To help you understand, we have implemented a miniature version of a 6-axis robot arm.)
    아두이노 우노 기반으로 6개 모터를 기반으로 DC 컨버터와 스텝 모터로 모터 출력을 구현하였습니다.
    (Based on the Arduino Uno, the motor output is implemented with a DC converter and a stepper motor based on 6 motors.)
